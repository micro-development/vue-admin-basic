# ITTLR 基础模块

> 管理端 基础模块，包含：账户中心。

## 启动项目

```bash
it-md install # 必须在仓库根目录执行
yarn install
npm run serve
```

浏览器打开访问：[http://localhost:8002](http://localhost:8002)

当收到更新模块通知时，执行一下命令更新：

```bash
it-md update # 必须在仓库根目录执行
```
